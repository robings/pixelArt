Help

Choose foreground and background colour using the colour pickers. To quickly choose a foreground colour, click on one of the colours in the palette.

To change the colour of a single cell, click the cell in the grid you want to change. It will change to the selected foreground colour. Hold down Option (Mac OS) or ALT (Windows) whilst clicking to change the cell to the selected background colour.

Holding down the left mouse button and moving the mouse around the grid will change the cells you roll over to the selected foreground colour. Hold down Option (Mac OS) or ALT (Windows) whilst holding down the left mouse button to change the cells you roll over to the selected background colour.

Click Random Art to have a random piece of art created where each cell in the grid is filled with a random colour. Select Totally random for every cell to be random. Otherwise there is a random chance that a cell next to another cell will be made the same colour. Select Random Art Slideshow and select a delay in milliseconds to have the random art change each time the delay has elapsed. To stop a random art slideshow, untick Random Art Slideshow, and click Random Art again.

To save a drawing, you can download a JSON file representing the art by clicking Download Drawing JSON. To restore the drawing, open the downloaded file in a text editor, select all, copy and paste it into the text input. Then click Restore Pasted Drawing.
